Project

CHIROVOX is an accessibility-focused project created by FOT Intake 41 ICT students at KDU. It helps translate sign language to spoken text using an on-device model and provides learning resources and a searchable dictionary. The camera page uses a trained model (when available) with fallbacks so the interface remains usable across devices.

Note: The live camera detection works best on modern browsers (Chrome/Edge/Firefox). On mobile devices you may need to serve the site over HTTPS — use the provided dev tunnel (ngrok/localtunnel) when testing on a phone.
